# jubilant-couscous — Cheerful Appreciation

A static landing page for **Cheerful Appreciation** — create and share heartfelt messages and certificates for the people who make a difference in your life.

## 🌐 Live site

**[https://kfaist.github.io/jubilant-couscous/](https://kfaist.github.io/jubilant-couscous/)**

## Structure

```
index.html   – Main HTML entry point (landing page)
styles.css   – Full stylesheet with design tokens, layout, and responsive styles
app.js       – JavaScript entry point (mobile nav toggle)
.github/
  workflows/
    deploy.yml – GitHub Actions workflow that deploys to GitHub Pages
```

## Getting started

Open `index.html` in a browser to run the app locally, or visit the [live site](https://kfaist.github.io/jubilant-couscous/) after changes are merged to `main`.
